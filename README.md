# 👨‍💻 Sobre Mim - Portfólio Inicial

Bem-vindo ao repositório do meu projeto "Sobre Mim"! Este projeto marca um dos meus passos iniciais na jornada para me tornar um programador, aplicando os conceitos fundamentais de desenvolvimento web.

## 🎯 Sobre o Projeto

Esta é uma página web estática desenvolvida para compartilhar um pouco sobre quem eu sou. Aqui, conto um pouco sobre minha experiência coordenando Compras e PCP, minha paixão por aventuras ao ar livre e, o mais importante, minhas metas de vida e carreira para os próximos meses.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Utilizado para a marcação e estruturação semântica de todo o conteúdo (aplicando tags de boas práticas como `figure`, `figcaption`, listas ordenadas e não ordenadas).

## ✨ Funcionalidades e Destaques

* **Estrutura Semântica:** Código limpo e focado em acessibilidade e boas práticas de SEO.
* **Links Inteligentes:** Contatos configurados não apenas para redes sociais, mas com chamadas diretas para o telefone (`tel:`) e e-mail (`mailto:`).
* **Listas e Hierarquia:** Organização clara de metas de curto a médio prazo.

## 🛠️ Como visualizar o projeto

É muito simples rodar este projeto na sua máquina:

1. Faça o download ou clone este repositório.
2. Certifique-se de que o arquivo da imagem está no caminho correto (`./img/Foto Perfil Ruben.jpg`).
3. Dê um duplo clique no arquivo `index.html` para abri-lo no seu navegador padrão (Chrome, Edge, Firefox, etc.).
